Frequently Asked Questions

App won't start

Please update to the latest version and restart your device. If the issue persists, contact us using the email below.

Where is my data stored?

All schedule data is stored locally on your device. Cloud synchronization is not currently supported.

Can I send feedback?

Yes. Suggestions and feature requests are always welcome.

When reporting a bug

Please include the following information if possible:

  • Device model
  • Android version
  • Family Scheduler version
  • Description of the issue
  • Error message (if available)
